# simple-eshop-cart
# Simple E-shop Cart System

Konzolová aplikace simulující logiku přidávání produktů do nákupního košíku e-shopu. 

## Popis funkčnosti
Aplikace je postavena na objektovém přístupu a skládá se ze dvou hlavních entit:
1. **Třída `Produkt`**: Reprezentuje konkrétní zboží. Uchovává informace o názvu, ceně za měrnou jednotku a počtu kusů.
2. **Třída `Kosik`**: Slouží jako kontejner pro vybrané produkty. Obsahuje metody pro manipulaci s obsahem.

## Klíčové vlastnosti
* Přidávání objektů typu `Produkt` do nákupního košíku.
* Práce s poli/seznamy (list) uvnitř třídy košíku pro uchování vybraných položek.
* Metoda pro vypsání kompletního obsahu košíku (výpis položek a jejich množství).

## Technologie
* Python 3
* OOP (tvorba a interakce instancí)
